# Algoritmos e Estruturas de Dados II 🐍

Este repositório contém as implementações, estudos e exercícios desenvolvidos durante a disciplina de **Algoritmos e Estruturas de Dados II**. O foco principal é a exploração de estruturas não-lineares, algoritmos de otimização e análise de eficiência, utilizando a linguagem **Python**.

---

## 📌 Objetivos do Repositório
* Implementar estruturas de dados complexas (Árvores, Grafos, Hash).
* Estudar algoritmos de ordenação avançada e busca em grafos.
* Analisar a complexidade assintótica (Tempo e Espaço) de cada solução.
* Manter um histórico de evolução no domínio da linguagem Python.
